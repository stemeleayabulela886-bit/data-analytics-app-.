AI Data Analytics Platform 🚀
The Unified Intelligence Hub for Modern Data Scientists
"Why use five tools when you can build one that connects them all?"

This project is a high-performance, full-stack data analytics platform designed to bridge the gap between static BI reporting and active AI intelligence. It integrates the robust data engineering principles of Microsoft Fabric, the visual clarity of Power BI/Tableau, and the predictive power of Generative AI into one seamless workspace.

🌟 Why I Built This
After mastering industry-standard tools like Excel, Power BI, Tableau, and Microsoft Fabric, I noticed a significant friction point: the "Integration Fatigue." Moving between raw SQL data, visualization dashboards, and predictive models required too many tab-switches and export/import cycles.

I built this platform as a self-learning challenge to prove that a unified environment—where raw data becomes an AI-powered insight in seconds—is the future of data science.

🔥 Key Features (The Differentiators)
🤖 AI Command Board
Unlike standard BI tools where AI is a plugin, this platform features a built-in natural language interface. Users can "talk" to their data to modify UI components, filter views, or calculate complex metrics on the fly.

📈 Predictive Playback
Move beyond reporting what happened. The Predictive Playback slider allows users to simulate market growth or variable shifts, using backend AI forecasting to visualize potential future outcomes in real-time.

⚡ Streamlined Data Lifecycle
Bypassing the standard "Export/Import" nightmare. This app handles the journey from raw CSV/SQL ingestion to interactive KPI generation within a single React/Python ecosystem.

🛠️ The Tech Stack
Frontend: React.js, Tailwind CSS (Modular, component-driven architecture)

Backend: Python (Data processing and predictive logic)

Database/Engineering: SQL & Microsoft Fabric integration principles

Visualization: Custom charting engine designed for speed and interactivity
